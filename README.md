# Mathematics Game

## Project Overview
The Mathematics Game is an interactive and educational game designed for players of all ages to improve their mathematical skills through fun and engaging challenges.

## Features
- **Multiplayer Support**: Compete with friends or family.
- **Various Difficulty Levels**: Tailored questions for beginners, intermediates, and experts.
- **Instant Feedback**: Players receive immediate feedback on their answers.
- **Leaderboard**: Track your progress and compare with others.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/babyG23/Mathematics-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Mathematics-game
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the game, run:
```bash
npm start
```
